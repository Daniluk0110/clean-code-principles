# SOLID ✅

Добро пожаловать в раздел SOLID. Здесь будут реальные примеры из бизнеса на PHP 8.4+ со строгой типизацией.

## Что будет внутри
- S: Принцип единственной ответственности
- O: Принцип открытости/закрытости
- L: Принцип подстановки Барбары Лисков
- I: Принцип разделения интерфейсов
- D: Принцип инверсии зависимостей

## Формат примеров
- Было (Плохо) -> Стало (Хорошо)
- Реальные задачи: платежи, отчеты, нотификации, интеграции

## Статус
В разработке. Скоро примеры и разборы.
